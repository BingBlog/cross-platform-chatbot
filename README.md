# 跨平台 AI Chatbot 项目

基于大模型的智能聊天机器人，支持多平台部署，提供完整的对话体验和会话管理功能。

## 项目目的

学习跨平台开发技术，掌握不同平台的应用开发方法和最佳实践。

## 学习重点

- **桌面应用特性**：系统集成、快捷键、拖拽等
- **移动应用特性**：手势、触摸、传感器等
- **平台差异**：不同平台的交互模式和设计规范
- **代码复用**：桌面和移动端之间的代码复用策略
- **性能优化**：各平台的性能优化技巧
